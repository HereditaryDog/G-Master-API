# 引言

欢迎使用 `G-Master API` 接口文档。

`G-Master API` 是一个兼容多家上游模型服务的统一中转网关，支持 `OpenAI`、`Claude`、`Gemini`、`Responses`、图片、音频、视频等多种接口形式。

## 基本地址

- 站点地址：`https://gmapi.fun`
- OpenAI 兼容基址：`https://gmapi.fun/v1`
- 后台 / 用户接口基址：`https://gmapi.fun/api`

## 文档适用对象

- 需要快速接入模型调用的开发者
- 需要在线调试接口的测试人员
- 需要管理令牌、查看余额、检查调用状态的用户

## 鉴权说明

- 模型调用接口默认使用 `Bearer Token`
- 用户中心接口通常使用 `Session` 或 `Access Token`
- 部分用户接口需要额外传 `New-Api-User` 请求头

## 建议阅读顺序

1. `API 快速开始指南`
2. `在线调试说明`
3. `代理接口调用地址`
4. 具体接口分组
