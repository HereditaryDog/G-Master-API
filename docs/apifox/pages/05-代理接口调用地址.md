# 代理接口调用地址

## OpenAI 兼容接口

- 基址：`https://gmapi.fun/v1`
- 常用路径：
  - `GET /v1/models`
  - `POST /v1/chat/completions`
  - `POST /v1/responses`
  - `POST /v1/embeddings`
  - `POST /v1/images/generations`
  - `POST /v1/audio/transcriptions`
  - `POST /v1/video/generations`

## Claude 接口

- `POST /v1/messages`

## Gemini 接口

- `GET /v1beta/models`
- `POST /v1beta/models/{model}:generateContent`

## 视频接口

- `POST /v1/videos`
- `GET /v1/videos/{task_id}`
- `GET /v1/videos/{task_id}/content`
- `POST /kling/v1/videos/text2video`
- `POST /kling/v1/videos/image2video`

## 用户中心与令牌管理

- 基址：`https://gmapi.fun/api`
- 常用路径：
  - `GET /api/status`
  - `GET /api/models`
  - `GET /api/token/`
  - `POST /api/token/`
  - `PUT /api/token/`
  - `DELETE /api/token/{id}`
  - `GET /api/user/self`
  - `GET /api/data/self`
