# 各种插件 / 软件使用教程

## Codex / Claude Code / 各类 Agent 工具

如果工具支持 OpenAI 兼容协议，优先填写：

- `Base URL = https://gmapi.fun/v1`
- `API Key = sk-your-token`

## Cherry Studio

- 提供商类型：`OpenAI Compatible`
- Base URL：`https://gmapi.fun/v1`
- API Key：你的 `sk-` 令牌

## ChatBox

- 选择 `OpenAI API`
- 接口地址：`https://gmapi.fun/v1`
- 密钥：你的 `sk-` 令牌

## Cursor / Cline / Aider

如果支持自定义 OpenAI 地址：

- `OpenAI Base URL = https://gmapi.fun/v1`
- `OpenAI API Key = sk-your-token`

## Dify / N8N / 工作流平台

优先使用：

- API 类型：`OpenAI Compatible`
- Base URL：`https://gmapi.fun/v1`
- API Key：你的 `sk-` 令牌

## 常见问题

- 工具提示 `Invalid authorization header`
  说明 API Key 或 `Bearer` 格式有问题。
- 工具提示模型不存在
  说明当前令牌没有该模型权限，或模型名填写错误。
