# v1.0.0-rc.10-GM.2

发布日期：2026-06-10

## 重点变化

- 使用 Open Design 重做控制台操练场页面，桌面端改为配置、对话、调试三栏工作台。
- 对话区新增状态栏、模型/分组摘要和固定底部输入区，便于持续测试模型输出。
- 调试信息面板支持右侧常驻或隐藏，移动端通过浮动入口打开，避免挤压主对话。
- 新增操练场相关多语言文案，保持中文、繁体中文、英文、法文、日文、俄文和越南文界面一致。

## 说明

本版本只调整操练场前端体验，不改变现有模型调用、分组加载、请求发送、调试记录和 Gaster Code 账号中心协议。
