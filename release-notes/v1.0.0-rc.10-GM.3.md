# v1.0.0-rc.10-GM.3

发布日期：2026-06-10

## 重点变化

- 重做控制台令牌管理页面，新增指标摘要、紧凑工具栏、状态化筛选和更清晰的令牌表格层级。
- 重做控制台使用日志页面，新增请求、成功率、消耗和 Token 摘要，查询区域与日志明细更适合运营排查。
- 移动端令牌管理和使用日志改为卡片视图，避免小屏幕横向滚动，同时保留复制、查看、编辑、删除和日志详情入口。
- 新增统一的 `gm-data-ops-*` 控制台样式层，令牌与日志页面共享工作台布局、KPI 卡片和响应式行为。
