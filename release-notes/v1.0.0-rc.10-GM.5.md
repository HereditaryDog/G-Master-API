# v1.0.0-rc.10-GM.5

发布日期：2026-06-11

## 重点变化

- 继续优化 `/gaster-code` 产品介绍页，补齐浅色/深色模式的语义色覆盖，避免深色模式出现黑字贴深背景、浅色模式出现下载区标题过度断行。
- 收敛 Gaster Code 页面标题、正文和下载区排版尺度，让桌面端和移动端都保持产品页节奏，但不再紧凑或压迫。
- 为能力卡片、工作流步骤、下载卡片、隐私边界和 FAQ 增加分组 reveal 动效，滚动进入视野时更平滑，并保留 `prefers-reduced-motion` 降级。
- 保持 Gaster Code 公开 release-only 仓库事实不变：公开页只引导用户下载安装器、ZIP 更新包、updater 元数据和未签名 macOS 安装说明，不暴露私有源码仓库。
